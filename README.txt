Cara Run Test_LPSE.py:

Pertama, pastikan python 3 sudah terinstall di komputer;

Kedua, install packages yang diperlukan dengan mengetik kode sebagai berikut satu per satu di command prompt (untuk windows) atau terminal (untuk mac)
  pip install pandas 
  pip install selenium

Ketiga, download chromedriver di https://chromedriver.chromium.org/. CEK chrome version yang tersedia di komputer dengan masuk ke Google Chrome -> Help -> About Google Chrome. Versi yang terbaru adalah 104.x.xxxx.xx. Download zip file sesuai dengan spesifikasi komputer (linux, mac, atau windows). EXTRACT file-nya menjadi "chromedriver.exe", dan masukkan filenya ke directory/folder yg sama dengan file "Test_LPSE.py".

Keempat, run file "Test_LPSE.py" di command prompt atau terminal dengan change directory ke foldernya "Test_LPSE.py" dan "chromedriver.exe", lalu ketik:
  python Test_LPSE.py

Kelima, tunggu beberapa saat untuk proses scraping selesai.

Terima Kasih.
